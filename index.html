Home Work #3

  • git --version - версия git;
  • git init - создание репозитория через терминал;
  • git add . - добавление всех файлов;
  • git status - просмотр изменений;
  • git commit -m - фиксирует изменения и сохраняет их в историю версий;
  • git log - просмотр истории изменений;
  • git remote -v - информация про удалённые ветки;
  • git clone - создает новый репозиторий с заданным именем;
  • git push - загружает все изменения локальной ветки в удалённый репозиторий;
  • git branch - список именованных веток коммитов с указанием выбранной ветки
  • git checkout -b - создание новой ветки;
  • git checkout - позволяет перемещаться между ветками;
  • git config --global user.name "имя" - устанавливает имя, которое будет отображаться в поле автора у выполняемых коммитов;
  • git config --global user.email "имя" - устанавливает адрес электронной почты, который будет отображаться в информации о выполняемых коммитах.
